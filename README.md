# 👋 Hand Gesture Sliding Puzzle

A simple 3x3 sliding puzzle game built in Python. Instead of using a mouse or keyboard, you control the game by swiping your hand in the air in front of your webcam! 

Built using **OpenCV** and **MediaPipe**.

## ⚙️ Setup

1. **Install dependencies:**
   ```bash
   pip install opencv-python mediapipe numpy

2. Download the Model:
Download the hand_landmarker.task file from the MediaPipe website and place it in the same folder as this script.

## How to Run
Just run the Python file:
    python backend.py

How to Play
Make sure your webcam is on and your hand is visible.

Swipe Left, Right, Up, or Down to move the puzzle tiles.

Arrange the numbers from 1 to 8 to win!

Press ESC to quit.
